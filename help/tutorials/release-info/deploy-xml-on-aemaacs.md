<h2><a id="user-content-manual-deployment-via-cloud-manager-git-pipeline" class="anchor" aria-hidden="true" href="#manual-deployment-via-cloud-manager-git-pipeline"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Manual Deployment via Cloud Manager git pipeline</h2>
<p>For customers who purchased XML Documentation for AEMaaCS before 03/28/2022 (to be updated), please follow these deployment instructions:</p>
<ul>
<li>
<p>If you are starting fresh, you can replace the code automatically generated by cloud manager with the code from the below repo which has XML plugin already integrated: <a href="https://github.com/Adobe-TCS/XML-documentation-for-AEMaaCS" rel="nofollow">https://github.com/Adobe-TCS/XML-documentation-for-AEMaaCS</a></p>
</li>
<li>
<p>If you already have checked-in customizations in cloud manager git repo, you can refer to the below repo for the instructions on how to add XML plugin in your existing code: <a href="https://github.com/Adobe-TCS/DoX-Installer-for-AEMaaCS" rel="nofollow">https://github.com/Adobe-TCS/DoX-Installer-for-AEMaaCS</a></p>
</li>
</ul>
<h2><a id="user-content-deployment-via-cloud-manager" class="anchor" aria-hidden="true" href="#deployment-via-cloud-manager"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Deployment Via Cloud Manager</h2>
<p>If you are a customer who purchased XML Documentation for AEMaaCS after 03/28/2022 (to be updated), please follow these instructions to add XML Documentation to your AEM as a Cloud Service environment:</p>
<ul>
<li>Login to cloud manager</li>
<li>Edit the program for which you want to configure XML Documentation</li>
<li>Switch to "Solutions and Add-ons" tab</li>
<li>In the "solutions and add-ons" table, click on Assets</li>
<li>Select "XML Documentation" and press save</li>
</ul>
<p>You have successfully configured your program for automatic provisioning of XML Documentation solution.</p>
<p><a target="_blank" rel="noopener noreferrer" href="/AdobeDocs/experience-manager-xml-documentation-learn.en/blob/main/help/tutorials/release-info/assets/addon-configuration.png"><img src="/AdobeDocs/experience-manager-xml-documentation-learn.en/raw/main/help/tutorials/release-info/assets/addon-configuration.png" alt="Configuring XML Documentation solution" style="max-width: 100%;"></a></p>
<p>NOTE: To intall XML Documentation on any environment under the integrated program, you must run the pipeline assoicated with the environment. No additional configuration is required in your CM git codebase for installing XML Documentation.</p>
